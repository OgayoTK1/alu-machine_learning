# Linear Algebra Python Scripts

This repository contains Python scripts for practicing basic **array and matrix manipulation** using slicing and list operations. All scripts are written in Python 3 and are executable.

---

## Files

### 0-slice_me_up.py
- **Description:** Demonstrates array slicing.
- **Functionality:**
  - `arr1` contains the first two elements of `arr`
  - `arr2` contains the last five elements of `arr`
  - `arr3` contains the 2nd through 6th elements of `arr`
- **Requirements:** No loops or conditionals, exactly 8 lines.
- **Usage:**
  ```bash
  ./0-slice_me_up.py
